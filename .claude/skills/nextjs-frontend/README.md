# Next.js 16+ Frontend Skill

This skill provides comprehensive support for Next.js 16+ frontend development using Claude Code. It includes:

## Features

- **Project Setup**: Automated Next.js 16+ project creation with recommended defaults
- **App Router**: Complete documentation and examples for Next.js App Router
- **Component Architecture**: Guidance on Server vs Client Components
- **Data Fetching**: Modern Next.js 16+ data fetching strategies
- **API Routes**: Documentation for both App Router and Pages Router API routes
- **Caching Strategies**: Next.js 16+ caching APIs with cacheLife() and cacheTag()
- **Form Handling**: Next.js 16+ Form component implementation

## Usage

This skill can be used by Claude Code when working on Next.js 16+ frontend projects. It provides:

1. Quick project setup with `setup_nextjs_project.py`
2. Complete documentation for Next.js 16+ features
3. Best practices for component architecture
4. Examples for data fetching, API routes, and caching
5. Reference documentation for Next.js APIs

- App Router architecture
